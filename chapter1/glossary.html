Glossary

Code

Meaningful combinations of text and symbols that a web browser can read and display.

Node

Any device (computer, tablet, smartphone) connected to a network.

Network

Collection of two or more computers linked together to share resources and information.

Internet of things (IoT)

The term used to describe the ever-growing number of devices connected to a network.

Data Lines

Allow data to move from one computer to another.

Internet Backbone

Collection of high-speed data lines that link major computer systems located around the world.

Internet Service Provided (ISP)

Company that has a permanent connection to the Internet backbone.

World Wide Web (Web)

Service that provideds access to information stored on web servers.

Protocols

Hypertext Transfer Protocol (HTTP)

Rules for exchanging text, graphics, audio, video, and other multimedia files on the web.

File Transfer Protocol (FTP)

Exchange files from one computer to another over the Internet

Transmission Control Protocol/ Internet Protocol (TCP/ IP)

Used to tranfer data efficiently over the Internet by properly routining it to its destination.
