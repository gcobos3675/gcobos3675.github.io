Lesson 3 : Enhancing a Website with Images and Links

Adding Images to a Website

Images can be many things from phots, diagrams, drawings, to charts.

There are 4 different file formats:

    • GIF: (Graphic Interchange Format) Usually a short video animation or frame. No sound is included and the animation loops.
      PNG: (Portable Network Graophics) A still image that usually can hold up to either 256 or 16 million colors.
      JPG: (Joint Photographic Experts Group) Made specifically for digital photos. Supports up to 16.7 million colors.
      SVG: (Scalable Vector Graphics) Uses markup language to create 2D graphics, images, and animations. Created by the W3C Working Group.
