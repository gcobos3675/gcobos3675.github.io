Lesson 1 : Introduction to the Internet and Web Development

Doctype (!DOCTYPE html)

The very first line of the webpage to declare an HTML 5 webpage.

HTML (html lang="en", /html)

Marks the start and end of a webpage. lang="en" attribute is to define the language the webpage is using. The default must be English.

Head(head, /head)

Contains the webpage title and other information about the webpage.

Title (title, /title)

Sets the title of a webpage.

Meta (meta charset="utf-8")

Contain information about the webpage data. Unicode Transformation Format (utf) allows computers to display and manipulate text.

Body (body, /body)

Where all the text, images, and links will be displayed on the webpage.

Navigation (nav, /nav)

Marks the start and end of the navigation area at the top of the screen.

Paragraph (p, /p)

Marks the start and end of a paragraph.

Line Break (br)

breaks a line of text to the next line

Thematic/ Horizontal Break (hr)

Creates a single horizontal line across the screen.

Header (header, /header)

Defines the content at the top of the webpage.

Bold (b, /b)

Makes text in between "b" tags bold.
