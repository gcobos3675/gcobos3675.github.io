Chapter One

Chapter 1 is the introduction to HTML and a basic overview about the Internet, web sites, web servers, and web browsers. This is also where I learned how to create a basic HTML webpage. The main essentials for an HTML website are the DOCTYPE, html, head, and body.

Chapter Two

Chapter 2 is about using a template to make all the webpages of a website have the same look and feel. Here we also learn about the importance of file management so all files are organized for easy access. Once the template for the web page is finished, I can save the template with a different name (in this case index.html) so I do not have to repeat the whole webpage from scratch.

Chapter Three

Chapter 3 is about inserting images and links to make a webpage appear more vibrant and lively. We use div elements (like this one) to divide content of a webpage into different sections. To insert an image or link, we start with an anchor tag (a), followed by the "href" attribute that that has the name of the file or URL that either the link will go to when clicked or the image that will be shown in on the webpage.
